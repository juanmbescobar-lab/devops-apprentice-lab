## DevOps Apprentice lab

This project is an educational, production-style lab to reach high skills in:

## Dockers & docker compose
## Git & Github Workflows (branches, Prs, releases)
## CI/CD with GitHub Actions
## Security and supply-chain 

## Structure

- `apps/`       : application components (API, worker)
- `infra/local` : Docker Compose for local platform
- `.github/`    : GitHub Actions workflows and templates
- `docs/`       : decisions, runbooks, learning notes
